# 📘 Manual de Usuario
## Updater Administrativo - RTMyG Administración v52

---

## 📌 ¿Qué es esto?

El **Updater Administrativo** es una herramienta que te permite actualizar la aplicación **RTMyG Administración** de forma fácil y rápida.

```
┌─────────────────────────────────────────────┐
│                                             │
│    🖥️  RTMyG Administración v52            │
│                                             │
│         ⬇️  ACTUALIZAR                      │
│                                             │
│    🆕  Nueva versión disponible             │
│                                             │
└─────────────────────────────────────────────┘
```

---

## 🚀 Cómo Actualizar (Paso a Paso)

### Paso 1: Cerrar la aplicación

**IMPORTANTE:** Antes de actualizar, asegúrate de que la aplicación **RTMyG Administración** esté **CERRADA**.

```
    ╔═══════════════════════════════════╗
    ║                                   ║
    ║   ❌  CERRAR LA APLICACIÓN        ║
    ║       antes de continuar          ║
    ║                                   ║
    ╚═══════════════════════════════════╝
```

Si la aplicación está abierta, el actualizador te mostrará un error.

---

### Paso 2: Abrir el Updater

Busca en tu escritorio el acceso directo llamado:

```
    ┌────────────────────────────────┐
    │                                │
    │   📁 Updater Administrativo    │
    │                                │
    │      (Doble clic aquí)         │
    │                                │
    └────────────────────────────────┘
```

**Haz doble clic** para abrirlo.

---

### Paso 3: Esperar la carga

Se abrirá una ventana negra. Espera unos segundos mientras se cargan las versiones disponibles:

```
╔════════════════════════════════════════════════════════════╗
║       ACTUALIZADOR - RTMyG Administración                  ║
╚════════════════════════════════════════════════════════════╝

[1/5] Verificando que el programa no esté en ejecución...
      √ Programa no está en ejecución
[2/5] Descargando lista de versiones disponibles...
      √ Lista de versiones obtenida
[3/5] Procesando versiones disponibles...
```

---

### Paso 4: Seleccionar la versión

Verás un menú con las versiones disponibles:

```
╔════════════════════════════════════════════════════════════╗
║         VERSIONES DISPONIBLES (últimas 10)                 ║
╠════════════════════════════════════════════════════════════╣
║  [ 1] Versión 516 (15:51)     ◄── LA MÁS RECIENTE          ║
║  [ 2] Versión 516 (15:03)                                  ║
║  [ 3] Versión 515                                          ║
║  [ 4] Versión 514                                          ║
╚════════════════════════════════════════════════════════════╝

Selecciona una opción (1-10): _
```

#### 💡 Recomendación:
> **Siempre selecciona la opción 1** (la primera de la lista).
> Esta es la versión más reciente y actualizada.

**Escribe el número `1` y presiona `Enter`.**

---

### Paso 5: Esperar la descarga

El sistema descargará e instalará la nueva versión automáticamente:

```
[4/5] Verificando archivos...
      √ Backup creado: backup\RTMyG Administracion v52 old.exe
[5/5] Descargando versión 516 (15:51) desde el servidor...
      √ Descarga completada (45678912 bytes)
      Reemplazando ejecutable...
      √ Ejecutable reemplazado exitosamente

╔════════════════════════════════════════════════════════════╗
║                                                            ║
║   ✓ ACTUALIZACIÓN A VERSIÓN 516 (15:51) COMPLETADA         ║
║                                                            ║
╚════════════════════════════════════════════════════════════╝

Presione una tecla para continuar . . .
```

---

### Paso 6: Finalizar

Presiona **cualquier tecla** para cerrar el actualizador.

**¡Listo!** Ya puedes abrir la aplicación actualizada.

## ❓ Preguntas Frecuentes

### ¿Qué significa el número entre paréntesis?

```
Versión 516 (15:51)
         │     │
         │     └── Hora en que se subió (15:51 = 3:51 PM)
         │
         └── Número de versión
```

Si ves dos versiones con el mismo número pero diferente hora, elige la **más reciente** (la que está primero en la lista).

---

### ¿Qué hago si me aparece un error?

#### Error: "El programa está en ejecución"

```
╔════════════════════════════════════════════════════════════╗
║  ERROR: El programa está en ejecución.                     ║
║  Por favor ciérralo antes de actualizar.                   ║
╚════════════════════════════════════════════════════════════╝
```

**Solución:** Cierra la aplicación RTMyG Administración y vuelve a intentar.

---

#### Error: "No se pudo descargar"

```
╔════════════════════════════════════════════════════════════╗
║  ERROR: No se pudo descargar el archivo.                   ║
║  Verifica tu conexión a internet.                          ║
╚════════════════════════════════════════════════════════════╝
```

**Solución:** Verifica que tienes conexión a internet y vuelve a intentar.

## 📁 ¿Dónde queda el backup?

Cada vez que actualizas, el sistema guarda una copia de seguridad de la versión anterior en la carpeta `backup`:

```
📁 Carpeta del programa
│
├── 📄 RTMyG Administracion v52.exe      ◄── Versión actual
│
└── 📁 backup
    ├── 📄 RTMyG Administracion v52 old.exe      ◄── Backup 1
    ├── 📄 RTMyG Administracion v52 old 1.exe    ◄── Backup 2
    └── 📄 RTMyG Administracion v52 old 2.exe    ◄── Backup 3
```
